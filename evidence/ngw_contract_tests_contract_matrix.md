## Matrix Scope Lenses

Every owned row below is mapped across four required lenses:

1. **Module boundary** — which core/shell ownership seam must be protected.
2. **Runtime entrypoint boundary** — which startup/reload/list/call path must observe the contract.
3. **External contract boundary** — which public config/docs/MCP semantics are promised.
4. **Client/tool-list observable boundary** — what an MCP client or provider-list caller can observe.
